# 如何写好提示词
深刻理解一下提示词。 就是给大模型套缰绳。   提示词的本质是给大模型设限，设限的目的是为了大模型能更好的完成你给的任务，而不是毫无限制的自由发挥。 


做好提示词工程的最重要的前提就是你要有写好提示词的能力。 提示词都写不好，去讲提示词工程是没有意义的。 
就好比一个人经常给你还给你讲软件工程的理论讲多厉害，但是就写不好代码。这不是就扯淡。
